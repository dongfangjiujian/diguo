SUN认证JAVA程序员(SCJP)招生简章
　　　　　　　　　　　　　　　　　-----JAVA开发语言培训和认证赋予您开发的非凡才能
电脑报远程教育中心 　教学研究部
　　随着网络技术在我国的飞速发展和普及，网络软件开发人才在国家大力发展软件产业的今天尤为紧缺，根据美国伊文斯市场调查公司报告指出：在美国ＪＡＶＡ程序开发人员的平均年薪最高，高于持有Ｃ＋＋，Ｃ和ＣＯＢＯＬ程序开发人员，而且在人才市场上，据权威部门预测，到２００５年我国对ＪＡＶＡ技术人员的需求量将高达２０万人，事实上，国内目前大量企业对ＪＡＶＡ开发人员的需求和要求获得ＳＵＮ认证的ＪＡＶＡ程序员证书的人员急剧增加也正好应证了这一点，为适应这一市场发展的需求，经过认真总结两年来在全国开展网络远程教育的成功经验，我们决定在全国范围内开展ＳＵＮ认证的ＪＡＶＡ程序员远程教育，培养我国信息产业急需的ＪＡＶＡ软件开发人才，以适应中国大力发展软件产业对ＪＡＶＡ开发人才的迫切需求。

　　一、 主办单位

　　主办单位：SUN（中国）有限公司教育部

　　承办单位：电脑报远程教育中心，思达网校

　　二、 招生对象

　　凡具有DOS?Windows或有一定编程基础（C、BASIC、VB等）、英语水平的在职人员?各大专院校在校学生及要求掌握JAVA开发语言知识的人员均可报名参加SUN认证JAVA程序员远程教育培训，通过国际认证考试可获得SCJP国际认证证书。

　　三、 教材资料



课程名称 认证类别 教材及资料 建议学习时间 价格
SCJP(sun认证JAVA程序员） SUN SUN认证JAVA 2 程序员学习指南
JAVA 2参考大全
单机实验光盘一张
60天 880元

　　(1)SUN认证JAVA 2 程序员学习指南，建议学习时间40天；考试号：310-025，教材为英文版
　　 本课程覆盖了复习考试的所有主题，面向国际认证考试，主要内容有：如何编写代码，使对象适用于集合；使用循环的方法编写代码；使用if和switch语句编写代码；使用try和catch捕获异常；编写和发出异常；设计和实现紧密封装的类；转换线程状态；通过等待和通知的方法与对象进行通信；使用StringBuffer类。
　　本课程适用于熟悉C或C++的编程人员, 也适用于想参加Sun Java 认证考试的学员。

　　(2)JAVA 2参考大全，建议学习时间20天；教材为中文版
　　从介绍JAVA语言的基础知识入手，通过丰富的示例系统的介绍了JAVA语言基础知识、JAVA标准库及JAVA开发环境，让你通过简单到复杂的示例程序来逐步理解JAVA的精髓。 重点在讲解JAVA语言开发技术知识，是一本世界一流的程序设计大师Herbert Schildt的力作。

　　(3)单机实验光盘：该套光盘极大地解决了远程学习JAVA开发语言技术的核心问题----无法看到动态的程序开发和调试的过程（因为JAVA开发语言的学习如果只借助教材学习和无开发需求的情况下，个人想要学习好JAVA开发语言通常是较难的），在使用中文的课件和JAVA开发实验演示光盘时，在单机上就可以自由选择相应章节进行知识点的学习，同时借助生动的有声演示实验，你将全面和快速的掌握好JAVA语言开发技术。

　　四、学习时间

　　报名时间：随时报名，款到开课

　　学习时间：２个月(从开通帐号之日算起)。超过时间可向consultation@yesky.com申请延长帐号使用时间。

　　五、学习方式

　　先在天极网校注册，选课并缴费之后，我们将为您邮寄相应的学习专用书籍，并开通您的网上学习账号，同时我们将用E-Mail通知您的学习账号、密码和书籍邮寄的有关信息，您在收到学习书籍后即可开始远程学习，在您学习的同时可以享用我们为您提供的丰富网上学习资源，整个学习环节包括"网下自学主教材和辅助教材、网上学习中文课件、网上实验（本地模拟实验、自己搭建网络环境做实验）、在线答疑、网上模拟英文考试、到Vue或Sylvan授权的国际认证考试中心参加正式的联网考试"等，建议学习时间约６０天（每天学习2-3小时）。如果您有问题，尽可以参加我们每周专门时间的在线答疑或发E-Mail给我们的网校教师，我们将在24小时内为您作出满意的回复。

　　六、学习费用

　　SCJP：880元/人　（含资料费、学习费、邮寄费等）
　　交费请看：联系方式

　　(以上价格为平均价格，如有少许出入，恕不多退少补)

　　国际认证考试费：1250元/次 　（请看国际认证考试考点列表>>）
　　
　　七、考核及发证

　　电脑报远程教育中心目前已开通远程在线模拟考试本专业的课程，有条件的学员可以在网上学习完本专业的各门课程后，可先行参加网上免费的模拟测试，最后须到就近的VUE或SYLVAN授权的国际认证考试中心，申请缴纳考试费后参加联网认证考试，考试成绩通过方能获得SUN公司为您颁发的全球认证的JAVA程序员（SCJP）证书，该证书可作为各企事业单位JAVA程序设计、开发、管理人员上岗的依据，目前已成为各IT公司及相关企业争相竞聘的高层次JAVA开发热门人才。 考试流程请见>>
