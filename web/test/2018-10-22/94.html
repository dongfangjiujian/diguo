下面是我的准备过程和一些好东西。

先简单介绍一下Java认证考试

对于Java程序设计员，Sun推出两项认证：Sun Certified Java Programmer (SCJP)和Sun Certified Java Developer(SCJD)。 SCJP测验您的Java程序设计概念及能力，内容偏重于Java语法及 JDK的内容；SCJD则进一步测试您用Java 开发应用程序的能力，您必须先完成一个程序的设计方案，再回答与此方案相关的一些问题。

大家都是高人，不要去参加什么培训了，你要培训课程，网上找得到。SCJP的培训课程没什么大用。
SCJP基本上都是选择题，简答题最多1，2题，我没碰到。

具体信息可以到 http://www.sun.com.cn/education/sun-auth.html http://suned.sun.com/USA/certification/index.html去仔细看，考点，样题都可以找到，建议最先看考点。而且直到考试都应该时时看……

考点(Objectives)本地浏览，这个Objectives包括SCJP，SCJD，SCEA 。

在这个过程中，有任何疑问都可以看JLS(java 语言规范)，权威有用。 如果你对Java语言不了解，要从头学，推荐SUN网站上的教程，是E文的，中文的书推荐Core Java(Java核心技术，2本)、Think in Java中文版。
这儿有一个Cert的FAQ 和一个介绍。

我首先看的是公认的专为scjp考试写的优秀教程：Java2 Certification last minute tutorial ，

http://www.jchq.net/tutorial/introj2.htm，by Marcus Green，请大家注意先看看它的前面的介绍。

http://www.jchq.net/是个很不错的站点，其中它的3套mock test，考过的人都说是和真实题目最接近的，最好的，如果你在这3套题目能考到95%以上，那么实际考试中就很有可能得高分。

考过的人都建议最后做这3套题目，作为一个摸底。jchq教程的题目也很不错。

http://www.jchq.net/discus/最好SCJP论坛，Marcus Green 和 Roseanne Zhang 两个超级高手在上面
